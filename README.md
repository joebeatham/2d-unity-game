# 2d-unity-game
2D game built inside unity using C# featuring combat systems, basic metroidvania gameplay, and pathfinding algorithms. 
NPC pathfinding with a Breadth First Search (BFS) algorithm is within the files Assets->Scripts->NPCSeek.cs and Assets->Scripts->FriendlyA.cs. 
Enemy AI is found within Assets->Scripts->PracticeDummy.cs, Assets->Scripts->DummyAI.cs, and Assets->Scripts->BrokenRobotAttack.cs. 
The rest of the scripts control other gameplay, such as the player character, the camera, and the respawn after dying
Pictures and videos are also attached of the pathfinding and gameplay. 
The pathfinding shows the NPC travelling between ladders as waypoints for the BFS until it reaches the torch. 
The gameplay shows the player character jumping and air walking over a gap, getting attacked by an enemy, then killing that enemy.
